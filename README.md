# mahder05-lab-automation

🚀 Automation Lab: A personal lab to practice and build skills in DevOps and automation tools.

🛠️ Tools Used

	•	Ansible
	•	Docker
	•	Kubernetes
	•	Minikube
	•	Terraform
	•	GitHub Actions (CI/CD)
	•	AWX
	•	Helm
	•	Colima

Automation-Lab-Setup-Flow

     colima --> docker --> minikube --> k8s --> vault --> terraform --> awx --> github-actions CI/CD --> playbooks --> Automation Jobs --> ansible-cli

⚡ Purpose

	•	Practice automation
	•	Build DevOps workflows
	•	Learn containerization & orchestration
	•	Implement CI/CD pipelines
